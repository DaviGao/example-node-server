## pm2-node-server

##### Example node-server width pm2.

## develop

#### local

```
npm start
```

#### development

```
npm run dev
```

#### pm2

```bash
# list all running processes:
pm2 [list|ls|l|status]
# Display all apps logs
pm2 logs
# monitor memory and CPU
pm2 monit
# start/stop/reload/delete processes
pm2 start/stop/reload/delete app/pid/all
```

## production

#### pm2 online

```
npm run online
```
